The goal of this project is to explore the dataset, identify patterns, discover relationships among variables, and generate actionable insights through statistical analysis and visualization.

## Key Findings

* Detected important patterns and relationships
* Generated insights for decision-making

Exploratory Data Analysis helps transform raw data into meaningful information and supports data-driven decision making.
